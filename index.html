Data Science Cheatsheets

What are Cheatsheets?

Cheatsheets are concise reference guides that provide quick access to essential information, commands, or concepts. They are especially useful in data science and programming, where remembering numerous functions, parameters, and best practices can be challenging.

Machine Learning Cheatsheets

  • Scikit-learn Cheatsheet

    Quick reference for Scikit-learn, a Python library that provides simple and efficient tools for data analysis and machine learning, including classification, regression, clustering, and dimensionality reduction.

  • NumPy Cheatsheet

    Quick reference for Numpy, A powerful library for numerical computing in Python, providing support for large, multi-dimensional arrays and matrices, along with a vast collection of mathematical functions.

  • Statistical tests in Python cheatsheet

    Quick reference for Python statistical analysis guide covering t-tests, ANOVA, chi-square, correlation, regression, and non-parametric tests with code examples.

  • Machine Learning Algorithms Cheatsheet

    Quick reference for Linear Regression, Logistic Regression, Decision Trees, Random Forest, SVM, KNN, K-Means Clustering, DBScan, and Hierarchical Clustering.

  • PyCaret Cheatsheet

    A comprehensive guide to PyCaret, covering setup, data preprocessing, model training, evaluation, deployment, and advanced features for streamlined machine learning workflows.

  • XGBoost Cheatsheet

    Quick reference for XGBoost, a popular gradient boosting library

  • CatBoost Cheatsheet

    Essential information about CatBoost, a gradient boosting library with built-in categorical feature support

  • Dimensionality Reduction Cheatsheet

    Dimensionality reduction using UMAP (Uniform Manifold Approximation and Projection) and t-SNE (t-Distributed Stochastic Neighbor Embedding) transforms high-dimensional data into lower dimensions, facilitating visualization and revealing underlying patterns by preserving local and global data structures, respectively

  • Principal Component Analysis Cheatsheet

    PCA is a dimensionality reduction technique that finds the directions of maximum variance in high-dimensional data and projects it onto a lower-dimensional subspace.

  • Pytorch Cheatsheet

    Quick reference for Pytorch, one of the top deep learning library written in Python

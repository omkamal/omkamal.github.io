Aider Usage Tips

1. Add Relevant Files to Chat

Only add files that need to be changed. Too many irrelevant files can confuse the AI.

2. Break Goals into Steps

Tackle your project one step at a time. Adjust files in the chat as you progress.

3. Discuss Complex Changes First

Use /ask to plan complex changes before implementation.

4. Handling Stuck Situations

If Aider gets stuck:

  • Use /clear to reset the chat
  • Drop extra files with /drop
  • Switch models with /model
  • Take turns coding with Aider

5. Fixing Bugs and Errors

Share error outputs using /run or /test commands.

6. Providing Documentation

Share documentation through:

  • Pasting snippets
  • Including URLs
  • Using /read command

7. Interrupting and Inputting

Use Ctrl+C to interrupt. For multi-line input:

  • Paste directly
  • Use { and } for multi-line messages
  • Use Meta-ENTER for new lines
  • Use /clipboard to paste from clipboard
